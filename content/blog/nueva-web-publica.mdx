La nueva web pública de ZynxFox ya está disponible para que cualquier comunidad pueda entender qué ofrece el bot, cómo configurarlo y dónde pedir ayuda.

## Qué incluye esta primera versión

- Página principal renovada con foco en presentar el bot de forma clara.
- Wiki pública con guías de configuración y módulos principales.
- Catálogo de comandos accesible desde la web.
- Centro de soporte y acceso directo al Discord oficial.
- Términos y política de privacidad listos para consulta pública.

## Qué cambia para los administradores

La configuración principal sigue estando dentro de Discord. Si administras un servidor, el flujo recomendado sigue siendo usar <CommandBadge command="/config" /> para ajustar módulos, mensajes y canales.

<Callout variant="info" title="Importante">
La web no reemplaza la configuración desde Discord en esta etapa. Su objetivo es ayudarte a entender mejor cada función y encontrar ayuda más rápido.
</Callout>

## Qué sigue

Durante las próximas actualizaciones se irán añadiendo más capturas, ejemplos, tutoriales y entradas de novedades para que la web crezca junto al bot.
